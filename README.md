# <span dir="rtl">🚌 BusTimes</span>

<span dir="rtl">**BusTimes** הוא כלי שנועד לקצר ולפשט את תהליך בדיקת זמני האוטובוסים הרלוונטיים עבורכם בבוקר. במקום להיכנס לאפליקציות כבדות, הפרויקט מייצר דף HTML קליל ומהיר הנגיש מהטלפון.</span>

## 🚀 המטרה
קיצור זמן הבדיקה של זמני האוטובוסים על ידי הצגת הנתונים הרלוונטיים בלבד, בצורה מרוכזת וזמינה מיידית.

## 🛠️ איך זה עובד?
1. **שליפת נתונים:** המערכת מושכת נתונים רלוונטיים ממערכות משרד התחבורה.
2. **אוטומציה:** התהליך מתבצע באופן אוטומטי בכל בוקר באמצעות **GitHub Actions**.
3. **יצירת דף:** המערכת בונה דף HTML סטטי המותאם לצפייה מהירה מהנייד.
4. **ממשק משתמש:** בדף הסופי ניתן לבחור במהירות את קו האוטובוס המבוקש ואת תחנת המוצא הרלוונטית.

## ⚙️ הגדרות (config.ini)
ניתן להתאים אישית את הקווים והתחנות המופיעים בדף. המבנה של קובץ הקונפיג מבוסס על מזהי קווים וקודי תחנה:

```ini
[LINES] 
60א= 1
60 = 1
...

[STOP_CODES]
43496= 1
43334= 1
...
```
